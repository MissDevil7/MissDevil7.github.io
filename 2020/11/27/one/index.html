draft

这是草稿文章。

第一步：在gitbash中输入命令
hexo new draft newpaper
（这样会在source/_draft中新建一个newpage.md文件）
第二步：预览草稿命令
hexo server –draft
第三步：发表命令
hexo publish draft newpage
